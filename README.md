# mimzo-geo

Geo databases (geosite.dat / geoip.dat) for Mimzo VPN routing.
Served via jsDelivr CDN for reliable delivery (incl. RU networks).

- `geosite.dat` — custom domain categories (mimzo whitelist, ads, etc.)
- `geoip.dat` — IP ranges

CDN: https://cdn.jsdelivr.net/gh/urtenovcom/mimzo-geo@main/geosite.dat
